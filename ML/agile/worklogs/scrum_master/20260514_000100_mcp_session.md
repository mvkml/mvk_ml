### What the Team Learned This Session
- MCP separates **tool definition** (mcp_server) from **tool consumption** (agent)
- The agent (OpenAI) does not do the calculation — it only **decides** to call the tool
- Azure OpenAI requires deployment name, endpoint, and api_version — different from regular OpenAI
- Port conflicts on Windows need `netstat` + `Stop-Process` to resolve
- Always protect `.env` with `.gitignore` from day one
