# Learning Roadmap — Machine Learning

## Phase 1 — ML Foundations (Sprint 1)
- Data Preprocessing
- Feature Engineering
- Feature Selection
- Model Evaluation Metrics
- Cross-Validation
- Hyperparameter Tuning

## Phase 2 — Supervised Learning (Sprint 2–4)
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Gradient Boosting (XGBoost)
- Naive Bayes
- Ridge / Lasso Regression

## Phase 3 — Unsupervised Learning (Sprint 5–6)
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN
- PCA (Principal Component Analysis)
- t-SNE
- Autoencoders

## Phase 4 — Deep Learning (Sprint 7–9)
- Artificial Neural Networks (ANN)
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN / LSTM)
- Transfer Learning
- Transformers / Attention Mechanism

## Phase 5 — Reinforcement Learning (Sprint 10–11)
- Q-Learning
- Deep Q-Network (DQN)
- Policy Gradient Methods
